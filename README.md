## 2.- Información Técnica

### 2.1.- Software requerido

* STS o IntelliJ
* Postman o similares

### 2.2.- Tecnologías Utilizadas

* SpringBoot 3.4.4
* Java 17
* Maven

## 3.- Ejecución en ambiente local

1. Clonar el proyecto
    ```bash
    git clone https://github.com/WashiBH/user-api.git
    ```
2. Ejecutar el proyecto
    ```bash
    mvn spring-boot:run
    ```
